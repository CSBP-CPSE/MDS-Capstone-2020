Ontario COVID-19 Map

Instructions
  • Change the value type using the dropdown above the map.
  • Hover over a health region or long-term care home (represented as points) to view the name. Blue homes represent those without an outbreak while red represents either and active or inactive outbreak.
  • Select a health region or long-term care home to show the details.
  • Scroll in the map to zoom in and out.
Data sources: Data from Statistics Canada proximity data and health data.
Authors: Ngan Lyle, Sofia Bahtmutsky, Kaitlyn Hobbs, Shreeram Murali.
